# The-Farmer-Was-Replaced
****Мои скрипты для игры "The Farmer Was Replaced"****  
**Предупреждение:**  
Здесь вы не найдете продвинутых техник и методов решений таких вещей, как лабиринты, сортировки, змейки.  
С данными кодами можно пройти игру, но на место в рейтинге расчитывать не стоит.  
Возможно позже добавлю более продвинутые методы, с объяснениями решений.

Все скрипты писались по мере открытия возможностей и доделовались по ходу.  

  support.py - вспомогательный скрипт, нужен вообще везде. Импортируется внутри других скриптов. Просто так вызывать не надо.  
  
  main.py - основной скрипт, для посадки и сбора растений одним дроном.  
  
  asynchronous_plantin.py - версия main.py с использованием мульти-дронов.  
  
  pumpkins.py - скрипт для выращивания огромной тыквы на все поле. Одним дроном.  
  
  asynchronous_cactus.py - скрипт для посдаки и полной сортировки кактусов. С использованием мульти-дронов.  
  
  lft_v2.py - вспомогательный скрипт, для проходжения лабиринтов. Используется внутри labyrinth.py и labyrinth_asinc.py. Просто так вызывать не надо.  
  
  labyrinth.py - скрипт для прохождения лабиринта одним дроном.  
  
  labyrinth_asinc.py - скрипт для прохождения лабиринта мульти-дронами.  
  
  dynoZaurus.py - скрипт для прохождения дино-змейки одним дроном, по "умному"  
  
  stuped_dynoZaurus.py - скрипт для прохождения дино-змейки одним дроном, в тупую пробегая все поле. (Не так плох, как кажется, в размере 32х32 превосходит "умную дино-змею").
